# ASCII Dice

A lightweight Python terminal application that simulates rolling dice and displays the results using ASCII art.

## Features

- 🎲 Roll between 1 and 6 dice
- 🎨 Beautiful ASCII dice rendering
- 🔀 Randomized results
- 🖥️ Works entirely in the terminal
- 🐍 Built using only Python's standard library

## Requirements

- Python 3.10+

## Usage

Run the application:

```bash
python script.py
```

You'll be prompted to enter how many dice you want to roll:

```
How many dice do you want to roll? [1-6]
```

Example:

```
How many dice do you want to roll? [1-6]3
```
